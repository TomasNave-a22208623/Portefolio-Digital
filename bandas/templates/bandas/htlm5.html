Tipos de Seletores CSS e Combinadores

Tipos de Seletores

Seletor: Universal (*)

Usado: Sim

Comentário: Seleciona todos os elementos. Utilizado para aplicar uma fonte a todos os elementos.

Seletor: Tipo (elemento)

Usado: Sim

Comentário: Seleciona todos os elementos de um determinado tipo. Utilizado para estilizar todos os <p> numa página específica. Utilizei nas páginas das Bandas, dos Blogues e do Curso.

Seletor: Classe (.classe)

Usado: Sim

Comentário: Seleciona todos os elementos de uma determinada classe. Utilizado para aplicar estilos a vários elementos com classes específicas como .header-image, .button, entre outros.

Seletor: ID (#id)

Usado: Sim

Comentário: Seleciona um elemento único com um ID específico. Utilizado para aplicar estilos únicos a elementos específicos como #footer-text e #main-content.

Seletor: Atributo ([atributo])

Usado: Não

Comentário: Seleciona elementos com um atributo específico. Não foi utilizado em nenhuma página.

Seletor: Pseudo-classe (:pseudo-classe)

Usado: Sim

Comentário: Seleciona elementos num estado específico. Utilizado para estilizar links quando são "hovered" numa página específica.

Seletor: Pseudo-elemento (::pseudo-elemento)

Usado: Não

Comentário: Seleciona e estiliza uma parte específica de um elemento. Não foi utilizado em nenhuma página.

Combinadores de Seletores

Combinador: Descendente ( )

Usado: Sim

Comentário: Seleciona elementos que são descendentes de um elemento especificado. Utilizado para estilizar <p> dentro de <div> numa página específica. Utilizei nas páginas das Bandas, dos Blogues e do Curso.

Combinador: Filho (>)

Usado: Não

Comentário: Seleciona elementos que são filhos diretos de um elemento especificado. Não foi utilizado em nenhuma página.

Combinador: Adjacente (+)

Usado: Não

Comentário: Seleciona elementos que são imediatamente precedidos por um elemento especificado. Não foi utilizado em nenhuma página.

Combinador: General Sibling (~)

Usado: Não

Comentário: Seleciona elementos que são precedidos por um elemento especificado, não necessariamente imediatamente. Não foi utilizado em nenhuma página.
