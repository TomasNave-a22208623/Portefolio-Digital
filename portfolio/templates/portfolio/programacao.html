As Minhas Capacidades em Desenvolvimento Web

Esta pagina vai explicar as capacidades que obti ao desenvolver este projeto. Aqui está uma visão detalhada sobre cada uma das páginas:


  • Página do Portfólio: Esta página serve como uma vitrine digital do trabalho realizado durante o percurso acadêmico. Além de funcionar como um portfólio convencional, esta página atua como um hub central, proporcionando acesso fácil e intuitivo às demais páginas do projeto.
  • Página das Bandas: A página dedicada às bandas apresenta uma coleção detalhada de álbuns e músicas, oferecendo uma plataforma para explorar e apreciar a diversidade musical. Utilizando tecnologias como Python e Django, esta página demonstra competência na criação de aplicações web dinâmicas e interativas.
  • Página do Curso: Esta seção é uma fonte abrangente de informações sobre o curso de Engenharia Informática, abordando disciplinas, conteúdos programáticos e projetos desenvolvidos ao longo do curso. Com uma abordagem educacional estruturada, esta página visa enriquecer a compreensão dos visitantes sobre a área de estudo.
  • Página do Blog: A página do blog é um espaço interativo onde os usuários podem criar e compartilhar artigos relacionados à área de Engenharia Informática. Incorporando recursos avançados de autenticação e gerenciamento de permissões, esta página promove a troca de conhecimentos e experiências entre a comunidade acadêmica
  • Página de Meteorologia: Uma adição inovadora ao projeto, esta página descreve uma API desenvolvida a partir da API do IPMA, fornecendo informações meteorológicas de maneira acessível e envolvente. Demonstrando habilidades na integração de APIs externas e na criação de serviços personalizados, esta seção destaca a capacidade de inovação e adaptação do autor.

Além disso, o projeto inclui uma página de login que oferece aos usuários a possibilidade de acessar e interagir com todas as páginas do projeto. Com recursos de registro e recuperação de conta, juntamente com medidas de segurança aprimoradas, esta página garante uma experiência de usuário confiável e segura.


Tecnologicamente, o projeto faz amplo uso de Python, Django, HTML, CSS e JavaScript, proporcionando interfaces dinâmicas e responsivas que enriquecem a experiência do usuário em todas as páginas. Esta realização exemplifica o compromisso do autor com a excelência acadêmica e a inovação tecnológica, refletindo sua paixão pela Engenharia Informática.

Tecnologias Utilizadas

Python

Conteúdos:

- Sintaxe, operadores, controle de fluxo e tipos de dados
- Funções e módulos
- Gestão de arquivos e pastas
- Classes

Utilização:

Python foi a linguagem principal escolhida para o desenvolvimento deste projeto. Utilizou-se Python para uma variedade de tarefas, desde a lógica de negócios até a manipulação de dados. As principais áreas de aplicação incluem a criação de estruturas de dados personalizadas, gerenciamento de arquivos e pastas, bem como a implementação de funcionalidades específicas do projeto. A flexibilidade e a vasta gama de bibliotecas disponíveis em Python contribuíram significativamente para a eficiência e a robustez do projeto

Django

Conteúdos:

– Web frameworks
– Arquitetura MVT
– Urls, views e templates
– Linguagem template
– Modelação de base de dados, ORM
– Formulários
– Autenticação
– Implantação na cloud

Utilização:

Django foi o framework web escolhido para o backend deste projeto. Utilizando o padrão de arquitetura Model-View-Template (MVT), o Django ofereceu uma estrutura organizada e coesa para o desenvolvimento de aplicativos web dinâmicos. Desde a configuração inicial do projeto até a implementação de funcionalidades avançadas, como autenticação de usuários e modelagem de banco de dados usando o ORM do Django, este framework proporcionou uma base sólida e escalável para a construção de todas as páginas do projeto.

HTML e CSS

Conteúdos:

– HTML e seus elementos
– Formulários para envio de dados
– HTML5 e elementos semânticos
– CSS e seus seletores, para estilização de conteúdos
– Propriedades CSS
– Design responsivo e layouts dinâmicos
– Efeitos e animações CSS
– Bootstrap

Utilização:

HTML e CSS foram os alicerces para a criação das interfaces de usuário em todas as páginas do projeto. Utilizando HTML5 e elementos semânticos, juntamente com as propriedades avançadas de CSS, como seletores e layouts responsivos, conseguio se criar páginas web visualmente atraentes e facilmente navegáveis.

JavaScript

Conteúdos:

– Programação Orientada a Eventos
– Manipulação do DOM
– JSON e APIs
– Single Page Applications

Utilização:

JavaScript desempenhou um papel crucial na interatividade e na experiência do usuário em algumas partes do projeto. Por exemplo, na página do portfólio, implementamos um botão de alternância de modo escuro usando JavaScript, proporcionando aos usuários uma opção de personalização adicional. Além disso, a criação de um relógio dinâmico na aplicação demonstra o uso prático do JavaScript para manipulação do DOM e programação orientada a eventos.
